About This Application

This application provides real-time weather information for any city using the WeatherAPI. Enter a city name to get current weather details, including temperature, humidity, wind speed, and more.

The World Clock feature displays the current time for various countries across different time zones, helping you stay connected globally.

Built with HTML, CSS, and JavaScript, this project aims to deliver a seamless and user-friendly experience.
